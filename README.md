## What It Does

The Zyana Content Agent is a full-stack React application that acts as your AI-powered content strategist, script writer, engagement analyst, and brand outreach manager — all in one dark-mode dashboard.

Built for founders running a 90-day "building in public" content series targeting 10K followers and UGC brand deals.

---

## Features

**Script Generator** — Generate ready-to-film scripts for Talking Head, Screen Recording, Case Study, Educational, and Vlog formats. Voice input (STT) and TTS playback included.

**Video Studio** — Higgsfield AI integration for AI-generated video clips directly from the dashboard.

**Engagement Tracker** — Real metrics input + AI analysis + Recharts charts. Monthly goal tracking.

**Brand Outreach** — Pipeline manager for 8 pre-loaded prospects (n8n, Make.com, GoHighLevel, Bland AI, OpenAI...). AI-generates personalized outreach emails + LinkedIn DMs.

**Media Kit Builder** — AI-written media kit with editable rate card. Export as .txt.

**Agent Chat** — Conversational AI with active learning — extracts strategic insights every 4 messages.

## Quick Start

```bash
git clone https://github.com/Zyananaqiyahlk/Content-Agent-Studio.git
cd Content-Agent-Studio
npm install
npm run dev
# Opens at http://localhost:5173
```

Get your Anthropic API key at [console.anthropic.com](https://console.anthropic.com).
